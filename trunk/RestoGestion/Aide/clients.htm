Gestion des Clients

Cette fenêtre permet de modifier la table des clients

Elle permet 3 actions :


  • Ajouter un client
  • Supprimer un client
  • Modifier un client


Les plats présents en base s'affichent dans la grille au centre

Annuler vide les champs de saisie

Fermer ferme le fenêtre


Ajouter un client

Remplir les champs nécessaires :

  • Le nom (obligatoire, limité à 50 caractères)
  • Le prénom (obligatoire, limité à 50 caractères)
  • L'adresse (limité à 100 caractères)
  • Le code postal (limité à 50 caractères)
  • La ville (limité à 50 caractères)
  • Le numéro de téléphone (limité à 50 caractères)
  • Une remarque (limité à 100 caractères)

Ensuite cliquer sur Ajouter et le client apparaît dans la grille


Astuce : Il est possible de passer d'un champ à l'autre grâce à la touche tabulation ou à la touche entrée


Supprimer un client

Sélectionner un client dans la grille

Les champs se remplissent avec la sélection pour valider

Appuyer sur Supprimer

Une fenêtre de confirmation apparaît

Répondre Oui et le client disparaît de la grille


Modifier un client

Sélectionner un client dans la grille

Les champs se remplissent avec la sélection pour valider

Modifier le(s) champ(s) désiré(s)

Appuyer sur Modifier

Une fenêtre de confirmation apparaît

Répondre Oui et les modifications apparaissent dans la grille
