Options

Cette fenêtre, composée de 2 onglets, permet de modifier les options du logiciel


Onglet Général


Langue : Permet de choisir le fichier de langue utiliser par le programme.

La liste déroulante est remplie avec le nom des fichiers *.lng présents dans le répertoire Langues\


Taux de TVA : Défini le taux de TVA par défaut sur les commandes.

Si dans le fenêtre de commande, le taux de TVA n'est pas défini (égal à 0) c'est ce taux là qui sera utilisé.


Mot de Passe : En cochant Utiliser un mot de passe, on active une fenêtre de mot de passe au démarrage de l'application.

Le mot de passe est à définir dans la zone de saisie Mot de Passe (limité à 10 caractères)


Mot de Passe Effacement : En cochant Utiliser un mot de passe, on active une fenêtre de mot de passe lorsque l'on veut effacer une commande.

Le mot de passe est à définir dans la zone de saisie Mot de Passe (limité à 10 caractères)


Onglet Impression


Entête : L'entête est le texte qui apparaît dans la fenêtre de salle ainsi qu'en entête de la facture

Pour que l'entête ne déborde pas des cadres prévus, le texte ne doit pas dépasser du champ de saisie.


Logo : En cliquant sur Parcourir..., une boite de dialogue s'ouvre pour choisir une image.

Les formats supportés sont : *.bmp,*.gif,*.jpg,*.jpeg

La taille du logo est de 80x80 pixels, l'image n'est pas redimensionnée si elle est trop grande ou trop petite.




Valider enregistre les paramètres et ferme la fenêtre

Annuler ferme la fenêtre sans prendre en compte les changements
