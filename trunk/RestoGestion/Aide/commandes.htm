Commande

Cette fenêtre permet la saisie d'une commande pour une table

5 actions sont possible dans cette fenêtre :


  • Ajout d'un plat
  • Suppression d'un plat
  • Modification d'un plat
  • Impression d'une facture pour la commande
  • Effacer entièrement la commande


Les plats commandés s'affichent dans la grille centrale

Annuler vide les champs de saisie

Fermer ferme la fenêtre


Ajouter un plat

Remplir les 5 champs :

  • Le nombre de couvert (limité à 10 caractères)
  • Le code (obligatoire, limité à 10 caractères)

Ou

  • La désignation (obligatoire)

Si la code existe, la désignation est automatiquement affichée et vice versa

  • Le prix (obligatoire, décimal, 2 chiffres après la virgule)
  • La quantité (obligatoire,numérique)

Ensuite cliquer sur Ajouter et le plat apparaît dans la grille


Astuce : Il est possible de passer d'un champ à l'autre grâce à la touche tabulation ou à la touche entrée


Supprimer un plat

Sélectionner un plat dans la grille

Les champs se remplissent avec la sélection pour valider

Appuyer sur Supprimer

Le plat disparaît de la grille


Modifier un plat

Sélectionner un plat dans la grille

Les champs se remplissent avec la sélection pour valider

Modifier le(s) champ(s) désiré(s)

Appuyer sur Modifier

Les modifications apparaissent dans la grille


Imprimer la facture

Renseigner les champs TVA et Remise (en pourcentage) au besoin

Le montant se recalcule en fonction de la remise

Appuyer sur Imprimer pour lancer l'impression de la facture



Effacer la commande

Vide la grille de la commande ainsi que le nombre de couvert

Si l'option est cochée, un mot de passe sera demandé avant d'effacer

Une fenêtre de confirmation apparaît

Cette action historise la commande dans la base.
