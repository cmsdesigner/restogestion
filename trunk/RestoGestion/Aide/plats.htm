Gestion des Plats

Cette fenêtre permet de modifier la table de plats

Elle permet 3 actions :


  • Ajouter un plat
  • Supprimer un plat
  • Modifier un plat


Les plats présents en base s'affichent dans la grille au centre

Annuler vide les champs de saisie

Fermer ferme le fenêtre


Ajouter un plat

Remplir les 3 champs :

  • Le code (unique, obligatoire, limité à 10 caractères)
  • La désignation (obligatoire, limité à 100 caractères)
  • Le prix (obligatoire, décimal, 2 chiffres après la virgule)

Ensuite cliquer sur Ajouter et le plat apparaît dans la grille


Astuce : Il est possible de passer d'un champ à l'autre grâce à la touche tabulation ou à la touche entrée


Supprimer un plat

Sélectionner un plat dans la grille

Les champs se remplissent avec la sélection pour valider

Appuyer sur Supprimer

Une fenêtre de confirmation apparaît

Répondre Oui et le plat disparaît de la grille


Modifier un plat

Sélectionner un plat dans la grille

Les champs se remplissent avec la sélection pour valider

Modifier le(s) champ(s) désiré(s)

Appuyer sur Modifier

Une fenêtre de confirmation apparaît

Répondre Oui et les modifications apparaissent dans la grille
