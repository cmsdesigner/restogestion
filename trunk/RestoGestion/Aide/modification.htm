Modification de Table

Cette fenêtre est accessible en cochant l'option Modification de la Salle dans le menu.

Elle permet de déplacer et de modifier la taille des tables affichées.

Toutes les modifications sont effectués directement.


Nom permet de changer le libellé de la table

Gauche définit l'emplacement du bord gauche de la table

Haut modifie le bord la position du bord supérieur de la table

Hauteur agrandi verticalement la table

Largeur allonge horizontalement la table

Active rend la table active (cliquable) ou non

Supprimer détruit la table

Valider ferme la fenêtre ne prenant en compte les changements

Annuler retourne sur la fenêtre principale sans effectuer les modifications
